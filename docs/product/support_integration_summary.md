## Архитектура решения

### Компоненты системы поддержки
1. **Веб-интерфейс** (`app/static/`) - пользовательский интерфейс
2. **OllamaClient** (`app/ollama_client.py`) - обработка команд
3. **Support API** (`app/support_api.py`) - REST API эндпоинты
4. **Support RAG** (`app/support_rag.py`) - поиск по базе знаний
5. **CRM Manager** (`app/crm.py`) - управление пользователями и тикетами

### Поток данных
```
Пользователь → Веб-интерфейс → OllamaClient → Support API → Support RAG/CRM
```

## Команды для тестирования

### Через веб-интерфейс
1. Откройте http://localhost:8000
2. Используйте команды:
   - `/support Почему не работает авторизация?`
   - `/faq`
   - `/users`
   - `/tickets`
   - `/stats`

### Через консольный интерфейс
```bash
python test_support_commands.py
```
